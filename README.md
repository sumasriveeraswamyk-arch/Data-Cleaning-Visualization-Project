# Data Cleaning & Visualization Project

## Project Overview
This project demonstrates data cleaning, preprocessing, and visualization using Python.

### Features
- Handle missing values
- Remove duplicate records
- Detect outliers
- Generate visual reports
- Data storytelling using graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Structure

## How to Run

### Install Libraries
pip install -r requirements.txt

### Run Project
python main.py

## Output
The project generates:
- Sales by Category graph
- Profit Distribution graph
- Sales Contribution Pie Chart
